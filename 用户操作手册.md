# 药用植物病虫害管理系统 - 用户操作手册

## 📖 手册说明

本手册详细介绍了药用植物病虫害管理系统的使用方法，包括Web端和小程序端的操作指南。

**适用用户**: 管理员、专家用户、普通用户  
**系统版本**: v1.0  
**更新日期**: 2024年12月

---

## 🎯 系统概述

### 系统功能
- **植物管理**: 药用植物信息查询和管理
- **病虫害识别**: 病虫害信息和防治方案
- **农药查询**: 农药信息和使用指导
- **求助系统**: 专家咨询和问题解答
- **评论互动**: 用户评论和经验分享

### 用户角色
- **👨‍💼 管理员**: 系统管理、用户管理、数据统计
- **👨‍🔬 专家用户**: 内容管理、求助回复、专业指导
- **👤 普通用户**: 信息查询、求助提交、评论互动

## 🌐 Web端操作指南

### 1. 用户注册与登录

#### 1.1 新用户注册
1. 访问系统首页: http://localhost:8081
2. 点击"注册"按钮
3. 填写注册信息：
   - **用户名**: 3-20个字符，支持中英文
   - **密码**: 6-20个字符，建议包含数字和字母
   - **邮箱**: 有效的邮箱地址
   - **用户类型**: 选择"普通用户"或"专家用户"
4. 点击"注册"完成注册
5. 系统提示注册成功，跳转到登录页面

#### 1.2 用户登录
1. 在登录页面输入用户名和密码
2. 选择正确的用户类型
3. 点击"登录"按钮
4. 登录成功后跳转到对应的工作台

#### 1.3 忘记密码
1. 点击登录页面的"忘记密码"链接
2. 输入注册邮箱
3. 系统发送重置密码邮件
4. 按邮件指引重置密码

### 2. 普通用户功能

#### 2.1 植物信息查询
1. **浏览植物列表**
   - 进入"植物"模块
   - 查看植物卡片信息
   - 支持分页浏览

2. **搜索植物**
   - 在搜索框输入植物名称
   - 点击搜索按钮
   - 查看搜索结果

3. **筛选植物**
   - 选择药用部位筛选条件
   - 选择热度排序方式
   - 应用筛选条件

4. **查看植物详情**
   - 点击植物卡片
   - 查看详细信息：
     - 基本信息（名称、别名、科属）
     - 药用部位和功效
     - 生长环境和分布
     - 相关病虫害信息
     - 用户评论

#### 2.2 病虫害查询
1. **浏览病虫害列表**
   - 进入"病虫害"模块
   - 查看病虫害分类
   - 浏览病害和虫害信息

2. **查看病虫害详情**
   - 点击病虫害条目
   - 查看详细信息：
     - 症状描述
     - 发生规律
     - 防治方法
     - 推荐农药
     - 相关植物

3. **搜索病虫害**
   - 输入病虫害名称或症状关键词
   - 查看匹配结果

#### 2.3 农药信息查询
1. **浏览农药列表**
   - 进入"农药"模块
   - 查看农药分类
   - 浏览农药产品

2. **查看农药详情**
   - 点击农药条目
   - 查看详细信息：
     - 产品名称和成分
     - 适用范围
     - 使用方法
     - 注意事项
     - 安全间隔期

#### 2.4 求助功能
1. **发起求助**
   - 进入"求助中心"
   - 点击"发起求助"
   - 填写求助信息：
     - 问题标题
     - 详细描述
     - 上传问题图片
     - 选择相关植物
   - 提交求助请求

2. **查看求助状态**
   - 在"我的求助"中查看
   - 查看求助状态：
     - 待审核
     - 已发布
     - 已回复
     - 已解决

3. **查看专家回复**
   - 点击求助条目
   - 查看专家回复内容
   - 对回复进行评价

#### 2.5 评论功能
1. **发表评论**
   - 在植物/病虫害/农药详情页
   - 在评论框输入内容
   - 点击"发表评论"

2. **查看评论**
   - 浏览其他用户评论
   - 查看专家权威解答

3. **管理我的评论**
   - 进入"我的评论"
   - 查看评论历史
   - 编辑或删除评论

#### 2.6 个人中心
1. **个人信息管理**
   - 查看个人统计信息
   - 编辑个人资料
   - 上传头像

2. **修改密码**
   - 输入当前密码
   - 设置新密码
   - 确认密码修改

### 3. 专家用户功能

## 💡 使用技巧

### 1. 搜索技巧
- **精确搜索**: 使用完整的植物名称
- **模糊搜索**: 使用植物别名或部分名称
- **症状搜索**: 描述病虫害症状关键词
- **组合搜索**: 使用多个关键词组合

### 2. 筛选技巧
- **多条件筛选**: 同时使用多个筛选条件
- **排序优化**: 根据需要选择合适的排序方式
- **重置筛选**: 及时清除不需要的筛选条件

### 3. 求助技巧
- **详细描述**: 提供详细的问题描述
- **清晰图片**: 上传清晰的问题图片
- **准确分类**: 选择正确的问题分类
- **及时反馈**: 对专家回复及时反馈

### 4. 评论技巧
- **有用信息**: 分享有价值的经验
- **礼貌用词**: 使用礼貌和专业的语言
- **图文并茂**: 配合图片说明
- **及时更新**: 根据新情况更新评论

## ❓ 常见问题

### 1. 登录问题
**Q: 忘记密码怎么办？**
A: 点击登录页面的"忘记密码"，通过邮箱重置密码。

**Q: 账号被锁定怎么办？**
A: 联系管理员解锁账号，或等待自动解锁。

### 2. 功能问题
**Q: 为什么搜索不到某个植物？**
A: 可能植物信息尚未录入，可以向专家求助或建议添加。

**Q: 上传图片失败怎么办？**
A: 检查图片格式和大小，确保网络连接正常。

### 3. 小程序问题
**Q: 小程序无法加载数据？**
A: 检查网络连接，重新进入小程序。

**Q: 如何绑定已有账号？**
A: 在"我的"页面点击"账号绑定"，输入用户名和密码。
